# design-pattern-exercise

使用 TypeScript 練習設計模式

# 首要安裝需求

- node ^13.12.0
- yarn ^1.19.1

# 指令

建構專案

```bash
yarn install
```

執行測試

```bash
yarn test
```

lint
```
yarn lint
```
